Quick Start Guide

Get up and running in three steps:

# Step 1: Clone and generate manifest git clone https://github.com/anshmajumdar121/context-optimizer cd context-optimizer python tools/context_mapper.py /path/to/your/project # Step 2: Add to Claude Custom Instructions Copy skill/claude-custom-instructions.md → Claude Settings # Step 3: Start coding efficiently Paste prompt/one-click-vertical-prompt.md in your chat ✅ Ready! You're now using 5-27x fewer tokens.

Claude Desktop Setup

Configure Claude Desktop for persistent optimization:

# 1. Open Claude Desktop Click your profile → Settings → Custom Instructions # 2. Paste the skill Copy contents of skill/claude-custom-instructions.md Paste into Custom Instructions field # 3. Save and restart All new chats will now use optimized workflows 💡 Pro tip: Upload CONTEXT_MANIFEST.md to a Project for auto-context

Claude Web Setup

Use Context Optimizer in your browser:

# 1. Go to claude.ai Navigate to claude.ai and sign in # 2. Add the workflow Click your name → Custom Instructions Paste skill/claude-custom-instructions.md # 3. Use one-click prompt for single chats Paste prompt/one-click-vertical-prompt.md at chat start 🎯 Works identically to Desktop version

Claude Projects (Recommended)

Get automatic context for specific projects:

# 1. Generate project manifest python tools/context_mapper.py /path/to/project # 2. Create Claude Project claude.ai → Projects → New Project Upload CONTEXT_MANIFEST.md to Knowledge # 3. Enable auto-context Toggle "Always use this project for my code tasks" 🚀 Best for: Long-term projects with multiple files
